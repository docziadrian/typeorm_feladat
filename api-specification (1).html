F1 2025 API specifikáció

Projekt: Formula 1 2025 szezonkezelő rendszer
Backend: Node.js, Express, TypeORM
Adatbázis specifikáció: MySQL 8 kompatibilis séma

1. Cél

Az API a Formula 1 2025 szezon adatainak kezelésére készült. A rendszer kezeli a csapatokat, versenyzőket, pályákat, versenyeket és futameredményeket. Az API REST-alapú, JSON adatcserét használ, és az Angular frontend számára biztosít CRUD és dashboard végpontokat.

7. Validációs és működési szabályok

  • Minden létrehozás és módosítás JSON payloadot vár.
  • Kapcsolt entitásokhoz az API ID-alapú feloldást használ: teamId, driverId, circuitId, raceId.
  • Nem létező rekord esetén az API 404 Not found választ ad.
  • Törlésnél a relációs szabályok érvényesülnek, például a használt csapat vagy versenyző törlése hibára futhat.
  • A race_result rekordok törlése a verseny törlésével kaszkádolható.

8. Import és üzemeltetés

  • MySQL/phpMyAdmin importhoz használható fájl: api/mysql/f1_2025_phpmyadmin.sql.
  • A backend jelenleg SQLite futtatási adatbázist használ, de a dokumentált séma MySQL-re készült.
  • Az alkalmazásport alapértelmezés szerint: 4444.
